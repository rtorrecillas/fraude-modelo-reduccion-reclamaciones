# Fraude Modelo Reduccion Reclamaciones

Presentacion interactiva en React/Vite para exponer la propuesta de deteccion de patrones MIT/MOTO con impacto en reclamaciones asumidas.

## Scripts

- `npm run dev`
- `npm run build`
- `npm run lint`
- `npm run preview`

## GitHub Pages

El proyecto queda preparado para desplegarse automaticamente en GitHub Pages mediante GitHub Actions.

Pasos en GitHub:

1. Crear el repositorio y subir la rama `main`.
2. Ir a `Settings > Pages`.
3. Seleccionar `Source: GitHub Actions`.
4. Hacer push a `main`.

El workflow publicara la carpeta `dist`.
